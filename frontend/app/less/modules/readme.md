# Modulos less

Aplicar estilo dos módulos que serão reutilizados
nesse diretorio. Alguns exemplos de módulos abaixo:

- modal
Modal seria um bom exemplo de modulo, pois caso o 
projeto cresça e tenha mais de uma sessão que utiliza
esse componente o código será reaproveitado

- formularios
Itens de formulario como input, select box, radio button,
etc também são bons exemplos de modulos, pois geralmente
quando a criação define um padrão é utilizado no 
projeto todo

Um módulo pode ser também uma área do projeto que 
é utilizada em várias sessões. Voce pode criar um modulo
e importar em todas as suas paginas pai. o modulo deve ser 
um arquivo com o nome do modulo. Ex.:

- box-post.less
